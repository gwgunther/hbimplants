## Notes

- **Before/after photos** require signed patient consent/release forms before the shoot
- **All patient-facing photos** should reflect demographic diversity (age, gender, ethnicity)
- **Image naming convention** for delivery: use the Shot ID as the filename prefix (e.g., `FAC-01-office-exterior.webp`)
- **Deliver two versions** of each image: full-resolution master + web-optimized (.webp, max 2000px wide)
- The website currently uses `.webp` and `.avif` formats — photographer should deliver in a format that can be easily converted (TIFF or high-quality JPEG masters are fine)
